## 🛠 How to Capture Screenshots

### Using iOS Simulator:
1. **Open iOS Simulator** (comes with Xcode)
2. **Select Device** - iPhone 14 Pro Max for largest size
3. **Navigate to Screen** - Go to the feature you want to showcase
4. **Capture Screenshot** - Device → Screenshot (Cmd+S)
5. **Save to Desktop** - Screenshots save automatically
6. **Repeat for Each Screen** - Dashboard, tasks, warranties, etc.

### Using Real Device:
1. **Connect iPhone** to Mac via USB
2. **Open Xcode** → Window → Devices and Simulators
3. **Select Your Device** → Take Screenshot
4. **Save Screenshots** - Higher quality than simulator

### Screenshot Editing:
1. **Use Sketch/Figma** - For adding text overlays
2. **Apple's Screenshot Templates** - Official device frames
3. **Consistent Styling** - Same fonts, colors, layout
4. **Export at 2x** - Retina resolution required

## 📊 Screenshot Performance Tips

### What Converts Well:
- **Clear Value Proposition** - Show immediate benefits
- **Real Use Cases** - Actual home maintenance scenarios
- **Before/After** - Show problem → solution
- **Social Proof** - Ratings, testimonials, user count
- **Premium Benefits** - Clear upgrade value

### What to Avoid:
- **Too Much Text** - Keep it scannable
- **Generic Stock Photos** - Use actual app interface
- **Cluttered Interface** - Clean, focused screens
- **Technical Jargon** - Use homeowner-friendly language
- **Empty States** - Show app with real data
